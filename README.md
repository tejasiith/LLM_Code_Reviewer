# LLM_Code_Reviewer
An AI-powered program that uses Large Language Models (LLMs) to analyze Python code in order to produce documentation, find bugs, explain functionality, and recommend enhancements.  Python, Gradio, and OpenAI LLMs were used to create an interactive web user interface.
---

## Features

- Explains what the given Python code does
- Detects potential bugs and logical issues
- Suggests performance and readability improvements
- Generates function-level documentation
- Simple browser-based UI using Gradio

---

## Tech Stack

- **Language:** Python
- **LLM:** OpenAI GPT models
- **Framework:** Gradio
- **Environment:** Virtualenv
